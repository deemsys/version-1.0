Alterando tipo da conta do usuário

Observação: Para utilizar esta ferramenta, você deve ter permissão de administrador.

Para alterar o tipo da conta do usuário, siga os passos:

  1. No seu Meu Site, clique na guia Usuários. Será exibida uma lista dos usuários do sistema.

  2. Em "Identificação do Usuário", clique no nome do usuário, cujo tipo de conta você deseja editar.

  3. No campo "Tipo:", altere para o tipo que deseja, e clique em Atualizar Detalhes.
