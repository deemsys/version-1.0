Resetando senha do usuário

Observação: Para utilizar esta ferramenta, você deve ter permissão de administrador.

Para resetar a senha do usuário, siga os passos:

  1. No seu Meu Site, clique na guia Usuários. Será exibida uma lista dos usuários do sistema.

  2. Em "Identificação do Usuário", clique no nome do usuário, cuja senha você deseja resetar.

  3. No campo "Criar Nova Senha :", digite a senha temporá para o usuário. Digite a mesma senha novamente no campo "Verificar Nova Senha:", e clique em Atualizar Detalhes.
