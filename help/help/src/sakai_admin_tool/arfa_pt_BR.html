Adicionando novo usuário aos usuários da ferramenta

Observação: Para utilizar esta ferramenta, você deve ter permissão de administrador.

Este documento é apenas para o Site Administrativo. Para mais informações sobre como adicionar participantes do seu Site, vejaAdicionando, editando, ou excluindo participantes.

Para adicionar um novo usuário aos Usuários da Ferramenta Administrativa:

  1. Em Meu Site menubar, clique Usuários.

  2. No começo da tela, clique em Novo Usuário.

  3. Preencha a Identificação do Usuário, Nome, Sobrenome, E-mail, e Criar e Verificar Nova Senha. No campo "Tipo:", preencha com o tipo de usuário apropriado:

    • Para permitir que usuários criem worksites, use registered .
    • Para evitar que usuários criem worksites, use guest , ou mantenha o campo "Tipo": em branco.

  4. Clique em Salvar Detalhes.
