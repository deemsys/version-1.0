Alterando o nome do usuário

Observação: Para utilizar esta ferramenta, você deve ter permissão de administrador.

Para alterar o nome do usuário, siga os passos:

  1. No seu Meu Site, clique na guia Usuários. Será exibida uma lista dos usuários do sistema.

  2. Em "Identificação do Usuário", clique no nome do usuário que você deseja editar.

  3. Edite o(s) campo(s) "Nome:" e/ou "Sobrenome:", e clique em Atualizar Detalhes.
