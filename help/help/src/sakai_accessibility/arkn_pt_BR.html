Problemas conhecidos de acessibilidade

Sakai contém uma série de questões de acessibilidade que ainda precisa ser resolvida:

  • JavaScript deve estar habilitado para que o software funcione.

  • Ferramenta de cabeçalhos são muitas vezes redundante devido a separação do título da página e dos frames de conteúdo.

  • Itens no menu drop-down menus no Gradebook, Profile, Preferences, Syllabus, Samigo, Presentation, Roster, Wiki, e Section carrega ao pressionar as teclas para cima ou para baixo. Pressione Alt-seta para cima ou Alt-seta para baixo para percorrer um menu.

Informe a Equipe de Acessibilidade do Sakai em elledge@msu.edu se você encontrar outros problemas. Use o assunto "Sakai Questões de Acessibilidade".

Além disso, note que é altamente recomendável usar o JAWS versão 8.0 e Window-Eyes versão 5.5 ou posterior, como o desempenho será significativamente melhor.

http://www.freedomscientific.com/downloads/jaws/jaws-downloads.asp

Verifique se há atualizações para Window-Eyes em:

http://www.gwmicro.com/Window-Eyes/Upgrades/

Dicas e truques para a navegação

Aqui estão algumas dicas de navegação:

  • Para se deslocar nas listas drop-down, pressione Alt-seta para baixo para abrir a caixa de seleção. Em seguida, pressione a seta para cima ou para baixo para visualizar as opções.

  • Depois de selecionar uma ferramenta, mova o cursor para o conteúdo da ferramenta com Alt-c.

  • Se você se perder, pressione Alt-c para retornar o cursor para o início da ferramenta.

  • Vá para os botões (ex., Cancelar) da página pressionando Insert-F5 para abrir para abrir o diálogo da lista de formulários. Então pressione Ctrl-End.

  • Use cabeçalhos (com a letra  h ) to navigate. O worksite e as ferramenta de seção, bem como o conteúdo da página, estão marcados com cabeçalhos.

  • Para ver todos os cabeçalhos de uma página, pressione Insert-F6, e, em seguida, as setas para cima e para baixo para usar a rolagem .

  • Pressione Insert-seta para baixo para repetir o texto

  • Se os links para as ferramentas, conteúdo ou worksites não funcionarem, pressione Insert-F6 para puxar a lista de cabeçalhos, e navegue usando cabeçalhos.

Como configurar o JAWS e Window-Eyes

Você pode executar a maioria das configurações recomendadas no Internet Explorer através do comando Insert-v, seguido pela letra relevante para os comandos MSAA (como  h  para cabeçalhos), e, em seguida usando a seta para cima ou seta para baixo.

Para acessar as configurações HTML pelo próprio JAWS (por exemplo, para alterar as configurações padrão para JAWS):

  1. Vá em Utilitários pressionando Alt-u .

  2. Selecione Gerenciados de Configuração do JAWS pressionando  c .

  3. Selecione Definir Opções pressionando  s .

  4. Escolha Opções HTML pressionando  h .

  5. Pressione Ctrl-Tab para mover entre as páginas.

Para acessar as configurações HTML dentro do Window-Eyes:

  1. Escolha Painel de Controle pressionando Ctrl-\ .

  2. Selecione o Menu Global pressionando Alt-l .

  3. Escolha o Menu Verbosity pressionando  v .

  4. Selecione MSAA pressionando  m .

Cabeçalhos

Use cabeçalhos para navegar. Links para worksites, ferramentas e conteúdos aparecem como cabeçalhos para facilitar a navegação. O conteúdo dentro das ferramentas também é estruturado hierarquicamente utilizando <h1>, <h2>, e <h3> tags.

JAWS e Window-Eyes habilitam cabeçalhos por padrão.

Frames

Por o frame de título ser frenquentemente repetido, é recomendável que você não anuncie eles para serem lidos por leitores de tela desnecessariamente. A configuração padrão para JAWS é não dizer os nomes dos frames.

Se você deseja configurar o JAWS para "Dizer nome do frame no início e fim":

  1. No Internet Explorer, pressione Insert-v para ir para o menu MSAA.

  2. Digite  f f , e, em seguida,  e  para executar.

Window-Eyes habilita o nome dos frames por padrão.

Teclas de acesso

Teclas de acesso seguem o padrão comercial do Reino Unido, sempre que possível, e são mnemônicas. Em um PC, use Alt invocar as teclas de acesso; em um Macintosh, use Ctrl.Teclas de acesso são anunciadas no título tag para cada link.

Portal baseado em teclas de acesso

  • Página de Acessibilidade: Alt-0. (Isso deve ser definido em Permissões pela sua instituição. Ele também está disponível na ferramenta de Ajuda em Acessibilitade).
  • Página de Ajuda: Alt-6
  • Passar para o conteúdo: Alt-c
  • Passar para a lista de ferramentas: Alt-l
  • Passar para worksites: Alt-w

Ferramenta específicas com teclas de acesso

Teclas de acesso estão disponíveis para a maioria das ferramentas com base em formulário.

  • Excluir, remover, ou cancelar: Alt-x
  • Editar ou revisar: Alt-e
  • Ajuda: Alt-h
  • Atualizar: Alt-u
  • Salvar: Alt-s
  • Ver ou visualizar: Alt-v

Atualização da tela

Os usuários controlam as atualizações da página. Para atualizar o conteúdo da página, pressione F5.

Para configurar o JAWS para desativar a atualização da página:

  1. No Internet Explorer, pressione Insert-vpara ir para o menu MSAA.

  2. Digite  r r , e, sem seguida,  e  para executar.

Para configurar o Window-Eyes, pressione Alt-Shift-m até ouvir "suspender todos".

Tags de Título

Tags de Título ajuda na compreensão das ações de botões e links fornecendo informações adicionais. Para configurar o JAWS para usá-los:

  1. Em JAWS, vá para Utilitários pressionando Alt-u .

  2. Vá para Gerenciar Configuração pressionando  c .

  3. Selecione Definir Opções pressionando Alt-s .

  4. Escolha Configurações HTML pressionando  h .

  5. Selecionar Gráficos Verbosity pressionando Ctrl-Tab Ctrl-Tab.

  6. Pressione Tab uma vez, e, em seguida, pressione a seta para cima até ouvir "Usar título".

  7. Pressione Enter.

Window-Eyes habilita as tags de títulos por padrão.

Conformidade

O objetivo dos desenvolvedores é continuar a fazer as aplicações em conformidade com a Seção 508 e WCAG 1.0 Prioridades Um e Dois, e tão útil quanto possível para as pessoas utilizar tecnologias assistivas. A maioria das ferramentas seguem as diretrizes de acessibilidade do W3C para títulos, formulários, tabelas, links e imagens, mas não incluem o script alternativos para JavaScript ou mensagens de usuários para pop-ups ou ações concluídas. As seguintes ferramentas incluem mais orientações:

  • Announcements
  • Chat Room
  • Discussion
  • Email Archive
  • Gateway
  • Home
  • My Workspace--Membership
  • My Workspace--Preferences
  • Page Wrapper
  • Permissions
  • Sakai News
  • Schedule
  • Site Info
  • Web Content
  • Worksite Setup

As ferramentas com JSF (Java Server Faces) não foram revisadas para incluir todos os elementos de acessibilidade. Estas incluem:

  • Ferramentas Administrativas (Users, Aliases, Sites, Realms, M, On-Line, Memory, Site Archive)
  • Gradebook
  • Profile
  • Preferences
  • Syllabus
  • Samigo
  • Presentation
  • Roster
  • Wiki
  • Section
  • Drop Box
  • Sections
  • Assignments
  • Resources

Compatibilidade com a tecnologia assistiva

Sakai foi testado com JAWS versão 8.0 e Window-Eyes versão 5.5. A conformidade foi avaliada utilizando A-Prompt, WebXACT e Validador W3C, mas não é garantido.
