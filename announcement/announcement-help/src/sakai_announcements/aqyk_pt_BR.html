Avisos: Visão Geral

O que faz

A ferramenta Avisos (Announcements) é usada para informar os participantes do site sobre itens de interesse atual. O Avisos poder ter vários anexos, como documentos e URLs.

Para ver os avisos de todos os sites aos quais você pertence, no menubar do Meu Workspace, clique em Avisos (Announcements).

Principais conceitos

Ordenar: Você pode ordenar os avisos por assunto, remente, acesso ou data.

Postar: Avisos são postados para um curso ou projeto do site. Os proprietários do site e os instrutores podem optar por ter um Aviso enviado automaticamente por e-mail a todos os participantes do site usando a configuração de notificação de "baixo" ou "alto". Para mais inforamções sobre o gerenciamento de e-mail, veja Ferramentas para gerenciar e-mail do site.

Considerações:

  • Os avisos são uma forma útil para postar uma notícia sobre uma importante mudança de prazos, horários de reuniões, ou locais de reunião.

  • Quando criar um aviso, você pode selecionar Mostrar para torná-lo visível, Ocultar para ocultá-lo, ou Especificar Datas para definir a data de início e a data final para visualizar a aviso.
