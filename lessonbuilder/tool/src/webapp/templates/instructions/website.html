ADDING CONTENT FROM PACKAGES SUCH AS CAMTASIA, WIMBA CREATE AND ARTICULATE

The "Add website" dialog allows you to upload content contained in a ZIP file. There is specific support for Camtasia, Wimba Create, and Articulate, but any ZIP file containing resources may be uploaded.

For formats that we recognize, including Camtasia, Wimba Create, and Articulate, an item will be created in Lesson Builder that will lead to the content.

This will also work for ZIP files containing a file index.html or index.htm at the top level. The Lesson Builder item will point to that file.

In other cases, Lesson Builder will upload the content, but it will not add an item. You may use "Add Resource" with "Choose Existing File" to locate and add any files that you want to appear in your Lesson Builder page.
