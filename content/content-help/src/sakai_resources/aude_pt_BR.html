Criar pastas de grupo utilizando a ferramenta Repositório

Observação: Para completar os procedimentos deste documento, você deve possuir um papel com as permissões necessárias. Para determinar seu papel, siga as instruções em Papéis dos Participantes. Para um entendimento básico de papéis e permissões, veja Papéis e permissões: Visão Geral.

A ferramenta Repositório permite que você crie pastas acessíveis somente por um grupo específico.

Criar um grupo

Para criar uma pasta de grupo, você deve primeiro criar um grupo de estudo. Para ajuda, veja Gerenciar grupos.

Criar uma pasta de grupo

Para adicionar uma pasta de grupo, siga estes passos:

  1. Na barra de menu, clique em Repositório. Próximo à pasta da qual você deseja criar uma nova pasta de grupo, a partir do menu Adicionar, escolha Criar Pastas.

  2. Insira um título.

  3. Clique em Adicionar detalhes para este item.

  4. Em "Disponibilidade e Acesso", selecione Exibir este arquivo somente para grupos selecionados. Marque a caixa de seleção próxima aos grupos que você deseja permitir acesso, e clique em Criar Pastas Agora.

    Observação:Se você não vê nenhuma escolha em "Disponibilidade e Acesso", você pode precisar de ajustar as permissões para a pasta raiz.

Alterar uma pasta existente para uma pasta de grupo

Para alterar uma pasta existente para uma pasta de grupo:

  1. No menubar, clique em Repositório. Ao lado da pasta que você deseja mudar para uma pasta de grupo, no menu Ações, escolha Alterar Detalhes.

  2. Em "Disponibilidade e Acesso", selecione Exibir este arquivo somente para grupos selecionados. Marque os grupos que você deseja permitir o acesso, e clique em Atualizar.

Gerenciar permissões para pastas de grupo

Uma vez que você criou uma pasta de grupo, você pode necessitar gerenciar permissões a fim de ajustar o nível de acesso para participantes do site. Para instruções detalhadas, veja Alterar as permissões para uma pasta em Repositório.
