Fazer upload de arquivos ao Repositório utilizando WebDAV com Mac OS X

WebDAV significa "Autoria e Versionamento Distribuídos baseados na Web" ("Web-based Distributed Authoring and Versioning") É um conjunto de extensões ao protocolo HTTP que permite ao usuários colaborar, editar e gerenciar arquivos em um servidor web remoto.

Você pode utilizar o WebDAV para fazer upload e acessar arquivos no Repositório do seu site, se você tiver as permissões necessárias.

Além disso, considere o seguinte:

  • Antes de você configurar um WebDAV, ao menos um objeto (arquivo ou pasta) deve existir na ferramenta Repositório de destino. Para instruções sobre adicionar arquivos sem utilizar WebDAV, veja Repositório.

  • Configurar WebDAV separadamente para cada site.

  • Salvar arquivos no seu site diretamente de um aplicativo (Word ou Excel, por exemplo) não funciona corretamente. Para melhores resultados, salve o arquivo primeiramente em seu computador, e então faça o upload para o seu site.

Observação: Nomes de arquivos que incluírem certos caracteres não serão aceitos. Para uma lista de caracteres não suportados, veja Caracteres não-suportados.

Mac OS X 10.4 (Tiger) ou posterior

Para utilizar o WebDAV para fazer upload e acessar arquivos no Mac OS X 10.4 (Tiger), siga estes passos:

  1. No Finder, pressione Command-k . (Alternativamente, a partir do menu Vá, escolha Conectar ao Servidor.)

  2. No campo "Endereço do Servidor:" , insira a URL que aponta para a ferramenta Repositório de destino.

    Entre em contato com o administrador do seu site para obter o formato da URL; ele provavelmente incluirá o siteID do seu site. Para instruções sobre como achar seu siteID, veja Determinar o siteID de seu site.

  3. Digite seu nome de usuário e senha, se solicitado.

Você pode agora arrastar e soltar pastas entre seu computador e os sites que você acabou de conectar.

Opcionalmente, você pode clicar no + (sinal de mais) à direita do endereço do servidor logo após você o inserir, e o servidor será adicionado à sua lista de servidores favoritos.

Quando você realiza o login, você pode escolher ter a senha lembrada. Se você fizer isto, você não será solicitado entrar com uma senha quando você entrar no seu Repositório pessoal ou nos Repositórios para qualquer site ao qual você tem acesso. Não faça isso ao menos que seu computador seja seguro contra o uso por outras pessoas.

Observação: Quando você utilizar o WebDAV para acessar repositórios utilizando este método, você pode notar que arquivos .DS_store estão sendo criados. A base de conhecimento da Apple fornece instruções para eliminar este comportamento: instruções para eliminar este comportamento.

Mac OS X 10.3 (Panther) e anterior

Utilizar o WebDAV para fazer upload e acessar arquivos no Mac OS X 10.3 (Panther) e anteriores requer um cliente externo chamado Goliath.

Observação: Goliath possui alguns problemas de performance e pode não funcionar 100% do tempo. Devido a isso, somente suporte limitado está disponivel caso você experimente problemas.

Você pode baixar uma cópia gratuita do Goliath em:

http://www.webdav.org/goliath/#download

Uma vez baixado o arquivo, clique duas vezes no mesmo para descomprimí-lo. Em seguida, clique duas vezes no arquivo .dmg resultante. Uma imagem de disco chamada GOliath 1.0.1 deve agora estar disponível. Siga estes passos:



  • Arraste e solte a parta Goliath contida na imagem para o seu disco rígido.

  • Abra o aplicativo Goliath.

  • No campo "Endereço do Servidor:", insira a URL que aponta para a ferramenta Repositório de destino.

    Entre em contato com o administrador do seu site para obter o formato do URL; ele provavelmente incluirá o siteID do seu site.

    Observação: Para instruções sobre como achar seu siteID, veja Determinar o siteID de seu site.

  • Digite seu nome de usuário e senha, se solicitado.

  • Clique na aba Configurações avançadas, e marque Desabilitar o recurso de codificação de Macintosh. E, em seguida, clique em OK.

    • Na primeira vez que você tentar conectar, você pode ser solicitado a aceitar um certificado para o site. Clique em OK caso solicitado.

    Você pode agora arrastar e soltar pastas entre seu computador e os sites que você acabou de conectar.
