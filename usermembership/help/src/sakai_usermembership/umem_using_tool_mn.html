Хэрэглэгчийн гишүүнчлэл:Хэрэгслийг ашиглах

Энэ хуудсан дээр:

  • Хэрэглэгчдийг хайх
  • Хэрэглэгчийн элсэлтыг олох

Хэрэглэгчдийг хайх

Эхний хэрэгслийн хуудас нь хайлтын зарим үзүүлэлтүүдийг танилцуулна. Тодруулж өгснөөр илүү нарийн хайлт хийж болно:

  • Хэрэглэгчийн төрөл: Хэрэглэгчийн үүргээр нь шүүх боломжийг олгодог.Байгаа бүх үүргийн жагсаалт гарна.
  • Хэрэглэгчийн удирдлага: Хэрэглэгчийн эх сурвалжийг сонгох боломжийг олгодог. Дотоод хэрэглэгчид нь дотоод өгөгдлийн санд хадгалагддаг. Гадны хэрэглэгчид нь гаднах нэвтрэх мехахизмаар хангагддаг ( UserDirectoryProvider интерфайсыг хэрэгжүүлэх).
  • Хайлтын нүд: Хэрэглэгчийн нэр, хэрэглэгчийн id болон email-д хайлт хийхийн тулд түлхүүр үгийг тодорхой зааж өгөх боломжийг олгоно.
'Search'-ыг дарснаар тодорхой зааж өгсөн өгөглийн дагуу хайлтын үр дүнг харуулна. Хайлтын үр дүн бүрийг үзэх бол дараах мэдээлэл өгөгдсөн байна:
  • Хэрэглэгчийн ID
  • Хэрэглэгчийн бүтэн нэр
  • Email
  • Төрөл (үүрэг)
  • Удирдлага (хэрэглэгчийн эх сурвалж)

.csv файл формат дахь хайлтын үр дүнг татах бол 'Export CSV'-ыг дар ( MS Excel-рүү импортлох боломжтой).

Анхаар: Гадны хэрэглэгчид зөвхөн одоо байгаа сайтад аль хэдийнэ элссэн тохиолдолд л жагсаалтад орно.

Хэрэглэгчийн элсэлтийг олох

хэрэглэгчийн хайлтын үр дүн дэх хэрэглэгчийн id линк дээр дарвал тэдний гишүүнчлэлийг шинэ хуудсанд харуулна. сайтын элсэлт бүрийг үзэхийг хүсвэл дараах мэдээлэл нээлттэй:

  • Сайтын нэр (сайтруу орох линкүүд)
  • Энэ сайтад элссэн групууд
  • Сайтын төрөл (курс, төсөл, г.м)
  • Сайтын нөхцөл ( FALL 2007 г.м)
  • Үүрэг (сайт дахь хэрэглэгчийн үүрэг)
  • Сайтын статус (нийтлэгдсэн, нийтлэгдээгүй)
  • Хэрэглэгчийн статус (идэвхитэй, идэвхигүй)
.csv файл формат дахь хайлтын үр дүнг татах бол 'Export CSV'-ыг дар ( MS Excel-рүү импортлох боломжтой).
