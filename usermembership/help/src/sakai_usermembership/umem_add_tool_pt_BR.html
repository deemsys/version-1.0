Participação do Usuário: Adicionar Ferramenta Participação do Usuário

Dica: Estas instruções requer privilégios de administrador.

Aicionando a ferramenta

A ferramenta Participação do Usuário só pode ser adicionada a um site utilizando a ferramenta 'Sites' Admin. Para adicionar a ferramenta, faça o login como Administrador e siga as instruções:

  1. Entre na ferramenta "Sites" e procure o site para adicionar a ferramenta.
  2. Clique no id do site.
  3. Desça e clique em 'Páginas' para adicionar uma nova página.
  4. Na próxima tela clique em "Nova página", escolha um nome para a página (normalmente 'Participação do Usuário') e, em seguida, clique em Ferramentas.
  5. Clique em 'Nova ferramenta ", selecione " Participação do Usuário (sakai.usermembership)' e clique em 'Salvar'para manter as alterações.
  6. Agora a ferramenta é exibida no site editado.

Conceder acesso

Se o usuário não tem privilégios de administrador, o acesso deve ser concedido seguindo as instruções Conceder acesso da ferramenta para outros usuários.
