Participação do Usuário: Conceder acesso da ferramenta para outros usuários

Dica: Estas instruções requer privilégios de administrador.

Conceder acesso a ferramenta

Uma vez que a ferramenta é adicionada a um site, o Administrador deve ser conceder a todos os outros usuários que irá utilizar a ferramenta de Participação do Usuário. Para conceder acesso a ferramenta, faça o login como administrador e siga estas instruções:

  1. Entre na ferramenta "Realms" e procure o site com a ferramenta Participação do Usuário. Você pode precisar usar a ferramenta 'Sites' para encontrar o site id correspondente.
  2. Clique no site realm para editar (deve ser semelhante a /site/xxxxx).
  3. Desça e clique sobre o papel adequado para editar(normalmente, esse deve ser o papel de 'maintain'). Esse deve corresponder ao papel do usuário no site.
  4. Marque a função 'usermembership.view' e clique em 'Salvar' para manter as alterações.
  5. Agora você pode usar a ferramenta.
